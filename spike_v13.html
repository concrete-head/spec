Spiking Neural Network v13

This is a single layer leaky integrate and fire (LIF) spiking neural network (SNN) that uses spike timing dependent plasticity (STDP). The leftmost column shows the input spikes, the topmost row is the output nodes, the main grid shows the connections.

With sufficient output nodes the network can learn any pattern.

Learnings / Changes / Outstanding Questions

  • By simply firing the node with the highest activation it causes multiple nodes to learn almost same thing. Each time a node fires it tunes itself more closely to the stimulus that just fired. Since the stimulus has a certain amount of natural variation we will end up creating one node for each of the infinite variations. A mechanism is needed to prevent this. This is solved by having a learning threshold that is some factor higher than the firing threshold. This causes nodes to learn a pattern only up to a certain point.

  • At 64x64 image size. Only approx 500 of 4096 (i.e. 12%) of inputs are needed determine what an object is. i.e. NodesToFire should be 500 and threshold needs to be less than that. The closer the threshold to 500 the more pixels have to match to get a node to fire
  • Still unsure what best decay rate is. Perhaps zero?
